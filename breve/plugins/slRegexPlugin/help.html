The slRegexPlugin is a simple plugin which adds some basic regular expression parsing to breve. This plugin defines two functions:
  • regex(regexString, searchString)

    Returns 1 if the regular expression regexString is found in searchString. Otherwise, 0 is returned.

  • split(regexString, searchString)

    Like the perl function split, splits searchString in to a list of substrings delimited by regexString. So, for example, split(":", "a:b:c") returns a list containing the elements "a", "b" and "c".

For more information on the regular expressions that this plugin matches, see man regex(3).
