The slURLPlugin is a simple breve plugin (Mac OS X only) which adds the ability to open URLs and download URL content into breve. This plugin defines the following functioins:
  • openURL(urlString)

    Sends a message to the Finder to open the specified URL in the user's browser of choice.

  • getURL(urlString)

    Downloads the specified URL and returns the contents as a string. Should only be used to retreive text, not to download files.

  • downloadURL(urlString, fileString)

    Downloads the specified URL and saves the contents to the file specified by fileString. If the file location is not an absolute path, it will appear in the default output directory as specified in the preferences. This could be used, for example, to download breve object files on the fly.
